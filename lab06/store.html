Preguntas a Responder

JavaScript

  • ¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?
    Es una buena práctica utilizar JavaScript para validar los inputs de los formularios antes de enviar los datos al servidor porque proporciona una mejor experiencia de usuario al dar retroalimentación inmediata sobre los errores en los datos ingresados. Esto ayuda a prevenir que se envíen datos inválidos al servidor, lo que puede causar problemas en el procesamiento de los datos y en la integridad de la base de datos. Además, al validar los datos en el lado del cliente se reducen las solicitudes al servidor, lo que puede mejorar el rendimiento y la eficiencia de la aplicación.
  • ¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?
    Se pueden saltar las validaciones hechas con JavaScript desactivando la ejecución del código JavaScript en el navegador, ya sea mediante la desactivación de JavaScript en la configuración del navegador o mediante la manipulación del código fuente de la página para eliminar o modificar las funciones de validación.
  • Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?
    Aunque es posible saltarse las validaciones de JavaScript, seguir utilizando JavaScript para validar los inputs de los formularios sigue siendo una buena práctica por varias razones:
    • Proporciona una mejor experiencia de usuario al dar retroalimentación inmediata sobre los errores en los datos ingresados.
    • Ayuda a reducir la carga en el servidor al validar los datos en el lado del cliente antes de enviarlos.
    • Aunque la validación en el lado del cliente puede ser el primer paso, aún se debe realizar una validación adicional en el servidor para garantizar la seguridad y la integridad de los datos. La validación en el lado del cliente sirve como una capa adicional de seguridad y mejora la experiencia del usuario, pero no debe considerarse como la única medida de seguridad.
